# 🧠 Prompt Principal

Este prompt é utilizado para gerar respostas inteligentes e automatizar tarefas com IA.

## 🎯 Objetivo

Criar respostas otimizadas com base no contexto fornecido pelo usuário.

## ⚙️ Exemplo de uso

"Crie um agente capaz de automatizar o envio de emails com base em dados fornecidos pelo usuário."

## 🚀 Aplicação

- Automação de tarefas
- Geração de conteúdo
- Integração com sistemas externos
